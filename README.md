📚 Simple Library Management System

A Java-based console application that simulates basic library operations.  
This project was developed as a final assignment for the Java Programming course, showcasing fundamental programming concepts such as arrays, methods, conditional statements, and user input handling.

---

🎯 Features
- Display all books with their status (*Available* / *Borrowed*).
- Search for books by name (supports partial matches).
- Borrow and return books with status updates.
- Delete books from the collection.
- Request new books not currently available.
- Support section with hotline numbers for complaints and technical issues.
- About section with project and author details.
- Exit option to close the program gracefully.

---

🛠️ Technologies Used
- **Java** (Core language)
- **Arrays** for data storage
- **Methods** for modular code organization
- **Conditional statements (if/else)** for program flow control
- **Scanner** for user input

---

🚀 How to Run
1. Make sure you have Java JDK installed
2. Download or clone this repository
3. Compile the file: javac LibraryManagementSystem.java
4. Run the program: java LibraryManagementSystem

---

📋 Menu Options
1. Display Books.
2. Search for a Book.
3. Borrow a Book.
4. Return a Book.
5. Delete a Book.
6. Request a Book.
7. Complaints and Support.
8. About the Library.
9. Exit.
